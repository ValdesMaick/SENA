Crear la estructura del código

Luego de elegir el tema principal de tu sitio Web es importante seguir los pasos que te mostraremos a continuación:

1 Paso

En el editor de código de tu preferencia crea la etiqueta '!DOCTYPE html' seguido de la selección del idioma en la estructuraras el sitio al igual que la apertura y cierre del 'head' y el 'body' como se muestra en la imagen, que es donde empezaremos a escribir nuestro código.

¿Como crear el Nav?

1 Paso

Crea las etiquetas de apertura y cierre del 'Nav'

2 Paso

Con la etiqueta 'a' definimos un enlace en el que los usuarios del sitio Web podrán hacer clic y dirigirse a una sección en específico de la página o direccionarlos a una página diferente.

3 Paso

Con el uso de el atributo 'href' haremos que el texto agregado aparezca en forma de hipervínculo.

A continuación, vemos el código HTML para nuestro 'Nav' y el resultado obtenido.

¿Como crear el Head?

1 Paso

Crea las etiquetas de apertura y cierre del 'HEAD'

2 Paso

Dentro de nuestro 'HEAD' con el fin de evitar problemas de SEO y la correcta visualización de caracteres especiales usaremos la etiqueta: meta charset UTF-8.

3 Paso

Con el uso de la etiqueta 'title' podremos nombrar nuestro sitio Web acorde al tema que hubiéramos escogido para la Web

4 Paso

Con el uso de la etiqueta: link rel="shortcut icon" href="img/icono.jpg" type="image/x-icon" estableceremos un icono para nuestro sitio Web el cual será característico y único.

A continuación, vemos el código HTML para nuestro 'HEAD' y el resultado obtenido.

¿Como crear el Header?

1 Paso

Crea las etiquetas de apertura y cierre del 'HEADER' para facilitar la creación de estilos le asignaremos un nombre de clase.

2 Paso

Con el uso de las etiquetas 'h1' y 'h2' crearemos un título y subtitulo respectivamente.

3 Paso

Con la etiqueta 'p' insertaremos un párrafo en el cual se tocara el tema a tratar.

4 Paso

Por último para insertar imágenes usamos: img src="img/VsCode1.png" alt="Programando..." class="imagen-abaut-us" width="800" height="400", donde el atributo 'src' lo usamos para acceder a la ruta de un archivo multimedia, 'alt' podremos darle una descripción a la imagen en caso de que al acceder al sitio Web no se carguen correctamente los archivos, le asignamos 'class' para insertar los estilos y por ultimo 'width' y 'height' para especificar la altura y ancho de la imagen.

A continuación, vemos el código HTML para nuestro 'HEADER' y el resultado obtenido.

¿Como crear el Main?

El elemento 'main' es el contenido principal de nuestro 'body' donde se estructura el tema principal de nuestro sitio Web.

1 Paso

Como son varios elementos lo que dispondremos en nuestra página usamos la etiqueta 'section' y le asignamos una 'class' con el fin de facilitar creación de estilos.

2 Paso

Con el uso de las etiquetas 'h1' y 'class' crearemos un título.

3 Paso

Con la etiqueta 'div' y 'class' se crea una divicion de la seccion e insertamos un subtitulo con la etiqueta 'h2' y un parrafo con la etiqueta 'p', si usaremos mas informacion en la pagina crearemos mas 'section' con la misma estructura logica.

4 Paso

Por último para insertar imágenes usamos: img src="img/VsCode1.png" alt="Programando..." class="imagen-abaut-us" width="800" height="400", donde el atributo 'src' lo usamos para acceder a la ruta de un archivo multimedia, 'alt' podremos darle una descripción a la imagen en caso de que al acceder al sitio Web no se carguen correctamente los archivos, le asignamos 'class' para insertar los estilos y por ultimo 'width' y 'height' para especificar la altura y ancho de la imagen.

A continuación, vemos el código HTML para nuestro 'MAIN' y el resultado obtenido.
